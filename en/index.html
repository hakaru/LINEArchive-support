Frequently Asked Questions

Check here if you need help

"Data not found" message appears

Please make sure that the messaging app is installed and that you have logged in at least once. The chat database is created after the first login.

"Access permission required" message appears

You may need to grant "Full Disk Access" permission in macOS security settings. Go to System Settings > Privacy & Security > Full Disk Access and add ChatArchive.

Is my data safe?

Yes. ChatArchive processes all data locally on your device. It does not perform any network communication and never sends data to external servers. Encryption keys are used only in memory and are never saved to disk.

What message formats are supported?

Text messages, images, stickers, videos, voice messages, files, location data, contacts, and other major message formats are supported. Some special message types may display differently.

Are images and stickers displayed?

Images and stickers cached by the messaging app can be displayed. If the cache has been cleared, thumbnails or placeholders will be shown instead. Due to app specifications, older image data may not be retrievable.

Is an internet connection required?

No, the app works completely offline. ChatArchive does not perform any network communication. All features are available even without an internet connection.
