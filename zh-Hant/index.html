LineArchive

安全備份與瀏覽您的 LINE 聊天記錄

macOS 14.0+ iOS 17.0+

主要功能

簡單且安全的 LINE 聊天記錄管理

🔓

LINE 加密資料庫解密

自動解密 macOS 版 LINE 的加密資料庫，讀取您的聊天記錄。

🖱

一鍵自動匯入

無需複雜設定。只需一鍵即可自動偵測並匯入 LINE 資料。

💬

LINE 風格聊天介面

以與 LINE 相同外觀的聊天介面瀏覽記錄，操作體驗令人熟悉。

🔍

全文搜尋

跨所有聊天記錄進行全文搜尋，快速找到過去的訊息。

📤

跨裝置資料共享

以 .linearchive 格式匯出和匯入，輕鬆將資料從 macOS 轉移至 iOS。

🔒

完全本機處理

所有資料處理皆在您的裝置上完成，不進行任何網路通訊。

使用方式

簡單 3 步驟瀏覽聊天記錄

1

啟動應用程式

啟動 LineArchive。請確認已安裝 macOS 版 LINE。

2

點擊「匯入」

點擊主畫面上的匯入按鈕。應用程式將自動偵測、解密並匯入 LINE 資料庫。

3

瀏覽聊天記錄

匯入完成後，即可在 LINE 風格介面中瀏覽和搜尋聊天記錄。

1

準備 .linearchive 檔案

將從 macOS 版 LineArchive 匯出的 .linearchive 檔案傳輸至 iOS 裝置（透過 AirDrop、iCloud 雲碟等方式）。

2

匯入檔案

點擊 .linearchive 檔案以在 LineArchive 中開啟，或從應用程式內的「匯入」選項中選擇。

3

瀏覽聊天記錄

匯入完成後，即可在 iPhone 上以 LINE 風格介面瀏覽和搜尋聊天記錄。

常見問題

遇到問題時請先查閱此處

顯示「找不到 LINE」訊息

請確認已安裝 macOS 版 LINE 應用程式，且已至少登入一次。LINE 資料庫會在首次登入後建立。App Store 版本及官方網站版本的 LINE 均支援。

顯示「需要存取權限」訊息

您可能需要在 macOS 安全性設定中授予「完整磁碟存取權」。請前往系統設定 > 隱私權與安全性 > 完整磁碟存取權，然後新增 LineArchive。

我的資料安全嗎？

是的。LineArchive 在您的裝置上本機處理所有資料，不進行任何網路通訊，絕不會將資料傳送至外部伺服器。加密金鑰僅在記憶體中使用，不會儲存至磁碟。

支援哪些訊息格式？

支援文字訊息、圖片、貼圖、影片、語音訊息、檔案、位置資訊、聯絡人等主要訊息格式。部分特殊訊息類型的顯示方式可能有所不同。

圖片和貼圖會顯示嗎？

macOS 版 LINE 已快取的圖片和貼圖可以顯示。若快取已被清除，將顯示縮圖或預設圖片。由於 LINE 的規格限制，較舊的圖片資料可能無法取得。

需要網路連線嗎？

不需要，此應用程式完全離線運作。LineArchive 不進行任何網路通訊。即使沒有網路連線，也可以使用所有功能。

隱私權政策

我們重視您的資料隱私

資料收集

LineArchive 不收集任何個人資訊或聊天資料。所有資料處理皆在您的裝置上本機完成。

網路通訊

本應用程式不進行任何網路通訊。無需網路連線即可完全運作。

加密金鑰處理

用於 LINE 資料庫解密的加密金鑰僅在處理期間於記憶體中使用，不會儲存至磁碟。

第三方分享

我們不會向任何第三方提供、分享或出售使用者資料。
