ดาวน์โหลด ซื้อใบอนุญาต

คุณสมบัติหลัก

จัดการประวัติแชท LINE อย่างง่ายดายและปลอดภัย

🔓

ถอดรหัสฐานข้อมูล LINE

ถอดรหัสฐานข้อมูลที่เข้ารหัสของ LINE บน macOS โดยอัตโนมัติ และอ่านประวัติแชทของคุณ

🖱

นำเข้าอัตโนมัติด้วยคลิกเดียว

ไม่ต้องตั้งค่าซับซ้อน ตรวจจับและนำเข้าข้อมูล LINE โดยอัตโนมัติด้วยคลิกเดียว

💬

หน้าจอแชทสไตล์ LINE

ดูประวัติแชทในหน้าจอที่มีรูปลักษณ์เหมือน LINE พร้อมประสบการณ์การใช้งานที่คุ้นเคย

🔍

ค้นหาข้อความแบบเต็ม

ค้นหาข้ามประวัติแชททั้งหมด ค้นหาข้อความในอดีตได้อย่างรวดเร็ว

📤

ส่งออกและนำเข้าข้อมูล

ส่งออกและนำเข้าในรูปแบบ .linearchive บันทึกและกู้คืนข้อมูลสำรองอย่างปลอดภัย

🔒

ประมวลผลบนอุปกรณ์ทั้งหมด

การประมวลผลข้อมูลทั้งหมดทำบนอุปกรณ์ของคุณ ไม่มีการสื่อสารผ่านเครือข่ายใด ๆ

วิธีใช้

ดูประวัติแชทได้ง่าย ๆ ใน 3 ขั้นตอน

1

เปิดแอป

เปิด LineArchive ตรวจสอบให้แน่ใจว่าได้ติดตั้ง LINE สำหรับ macOS แล้ว

2

คลิก "นำเข้า"

คลิกปุ่มนำเข้าบนหน้าจอหลัก แอปจะตรวจจับ ถอดรหัส และนำเข้าฐานข้อมูล LINE โดยอัตโนมัติ

3

ดูประวัติแชท

เมื่อนำเข้าเสร็จสมบูรณ์ คุณสามารถดูและค้นหาประวัติแชทในหน้าจอสไตล์ LINE ได้

คำถามที่พบบ่อย

หากพบปัญหา โปรดตรวจสอบที่นี่

แสดงข้อความ "ไม่พบ LINE"

โปรดตรวจสอบว่าได้ติดตั้งแอป LINE สำหรับ macOS แล้ว และได้เข้าสู่ระบบอย่างน้อยหนึ่งครั้ง ฐานข้อมูล LINE จะถูกสร้างหลังจากเข้าสู่ระบบครั้งแรก รองรับทั้ง LINE จาก App Store และจากเว็บไซต์ทางการ

แสดงข้อความ "ต้องการสิทธิ์การเข้าถึง"

คุณอาจต้องให้สิทธิ์ "การเข้าถึงดิสก์แบบเต็ม" ในการตั้งค่าความปลอดภัยของ macOS ไปที่ การตั้งค่าระบบ > ความเป็นส่วนตัวและความปลอดภัย > การเข้าถึงดิสก์แบบเต็ม แล้วเพิ่ม LineArchive

ข้อมูลของฉันปลอดภัยหรือไม่?

ใช่ LineArchive ประมวลผลข้อมูลทั้งหมดบนอุปกรณ์ของคุณ ไม่มีการสื่อสารผ่านเครือข่ายใด ๆ และไม่เคยส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก คีย์เข้ารหัสใช้เฉพาะในหน่วยความจำเท่านั้นและไม่เคยบันทึกลงดิสก์

รองรับรูปแบบข้อความใดบ้าง?

รองรับข้อความตัวอักษร รูปภาพ สติกเกอร์ วิดีโอ ข้อความเสียง ไฟล์ ข้อมูลตำแหน่ง รายชื่อติดต่อ และรูปแบบข้อความหลักอื่น ๆ ข้อความบางประเภทพิเศษอาจแสดงผลแตกต่างออกไป

รูปภาพและสติกเกอร์จะแสดงหรือไม่?

รูปภาพและสติกเกอร์ที่ LINE บน macOS แคชไว้จะสามารถแสดงได้ หากแคชถูกลบ จะแสดงภาพขนาดย่อหรือภาพตัวแทนแทน เนื่องจากข้อกำหนดของ LINE ข้อมูลรูปภาพเก่าอาจไม่สามารถเรียกคืนได้

ต้องเชื่อมต่ออินเทอร์เน็ตหรือไม่?

ไม่ต้อง แอปทำงานแบบออฟไลน์ได้อย่างสมบูรณ์ LineArchive ไม่มีการสื่อสารผ่านเครือข่ายใด ๆ ทุกฟีเจอร์สามารถใช้งานได้แม้ไม่มีการเชื่อมต่ออินเทอร์เน็ต

นโยบายความเป็นส่วนตัว

เราให้ความสำคัญกับความเป็นส่วนตัวของข้อมูลของคุณ

การเก็บรวบรวมข้อมูล

LineArchive ไม่เก็บรวบรวมข้อมูลส่วนบุคคลหรือข้อมูลแชทใด ๆ การประมวลผลข้อมูลทั้งหมดทำบนอุปกรณ์ของคุณ

การสื่อสารผ่านเครือข่าย

แอปนี้ไม่มีการสื่อสารผ่านเครือข่ายใด ๆ ทำงานได้อย่างสมบูรณ์โดยไม่ต้องเชื่อมต่ออินเทอร์เน็ต

การจัดการคีย์เข้ารหัส

คีย์เข้ารหัสที่ใช้ถอดรหัสฐานข้อมูล LINE ใช้เฉพาะในหน่วยความจำระหว่างการประมวลผลเท่านั้น และไม่เคยบันทึกลงดิสก์

การแบ่งปันกับบุคคลที่สาม

เราไม่ให้ แบ่งปัน หรือขายข้อมูลผู้ใช้แก่บุคคลที่สามใด ๆ
